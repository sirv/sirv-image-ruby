Installation

gem install sirv_image
require 'sirv_image'

Quick Start

url = Sirv::SirvImage.new('https://demo.sirv.com/image.jpg')
  .resize(width: 300, height: 200)
  .quality(80)
  .format('webp')
  .to_url

Constructor

Sirv::SirvImage.new(url_or_base, path = nil)

img = Sirv::SirvImage.new('https://demo.sirv.com/image.jpg')
img = Sirv::SirvImage.new('https://demo.sirv.com', '/image.jpg')

Resize

MethodDescription
resize(width: nil, height: nil, option: nil)Set dimensions and scale option (fit, fill, ignore, noup)
width(w)Set width
height(h)Set height
scale_by_longest(s)Resize by longest dimension
thumbnail(size = 256)Create a square thumbnail
 
img.resize(width: 400, height: 300, option: 'fill')

Crop

MethodDescription
crop(width: nil, height: nil, x: nil, y: nil, type: nil, pad_width: nil, pad_height: nil)Crop the image
clip_path(name)Apply a Photoshop clipping path
 
img.crop(width: 200, height: 150, x: 10, y: 20)
img.crop(type: 'face', pad_width: 50)

Rotation

MethodDescription
rotate(degrees)Rotate (-180 to 180)
flipFlip vertically
flopFlip horizontally

Format & Quality

MethodDescription
format(fmt)jpg, png, webp, optimal, original
quality(q)Set JPEG quality (0-100)
webp_fallback(fmt)Set WebP fallback format
subsampling(value)Set chroma subsampling
png_optimize(enabled = true)Enable PNG optimization
gif_lossy(level)Set GIF lossy compression

Color Effects

MethodDescription
colorize(color: nil, opacity: nil)Apply color overlay
colortone(preset = nil, color: nil, level: nil, mode: nil)Apply color tone
 
img.colortone('sepia')
img.colortone(color: 'ff5500', level: 80, mode: 'highlights')

Effects

MethodDescription
blur(v)Apply blur (0-100)
sharpen(v)Apply sharpening (0-100)
vignette(value: nil, color: nil)Apply vignette
opacity(v)Set opacity (0-100)

Text Overlay

text(content, **options)

Add text overlay. Call multiple times for layers.

OptionDescription
font_size:Font size px
font_family:Google Fonts family
color:Text color
opacity:0-100
outline_width:Outline width
outline_color:Outline color
background_color:Background color
align:left, center, right
position:center, north, south, etc.
position_x:X offset
position_y:Y offset
 
img.text('Hello', color: 'white', font_size: 24, position: 'center')
   .text('World', position: 'south')

Watermark

watermark(path, **options)

Add watermark. Call multiple times for multiple watermarks.

img.watermark('/wm.png', position: 'center', opacity: 50)

Canvas

canvas(width: nil, height: nil, color: nil, aspect_ratio: nil, ...)

img.canvas(width: 800, height: 600, color: 'ffffff')
img.canvas(aspect_ratio: '16:9')

Frame

frame(style: nil, color: nil, width: nil, rim_color: nil, rim_width: nil)

img.frame(style: 'solid', color: '333333', width: 5)

Other

MethodDescription
page(num)Select a PDF page

URL Generation

MethodDescription
to_urlReturns the final URL string
to_sAlias for to_url. Works with string interpolation
 
url = img.to_url
puts "Image: #{img}"

Full Examples

url = Sirv::SirvImage.new('https://demo.sirv.com', '/product.jpg')
  .resize(width: 800, option: 'fill')
  .crop(type: 'face')
  .format('webp')
  .quality(85)
  .sharpen(20)
  .watermark('/logo.png', position: 'southeast', opacity: 40)
  .text('Sale!', color: 'red', font_size: 32, position: 'north')
  .to_url
